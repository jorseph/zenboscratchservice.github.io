積木使用說明 Scratch Blocks

專業版 教育版

以下列出 Zenbo Scratch v1.2版 教育精簡版 所有積木的使用說明：

1. 設定 Zenbo 的 IP 變數, 你可以在 Zenbo Scratch 設定畫面上看到目前 Zenbo 的 IP.

2. 控制 Zenbo (前進 / 後退), 以速度 ( 慢 / 一般 / 快) 移動 ( 0.25 / 0.5 / 0.75 / 1.0 / 1.25 / 1.5 / 1.75 / 2.0) 公尺.

3. 控制 Zenbo 停止移動.

4. 控制 Zenbo 頭部向 (左 / 右/ 上 / 下) 轉 ( 0 / 15 / 30 / 45 )度, 此角度為絕對角度, 若已經為頭向上15度, 再下一次頭向上15度的指令則不會有反應.

5. 控制 Zenbo 向 (左 / 右)轉動身體 (0/ 15/ 30/ 45/ 60/ 75/ 90/ 105/ 120/ 135/ 150/ 165/ 180) 度.

6. 控制 Zenbo 持續 (停止/ 前進/ 左轉/ 右轉).

7. 控制 Zenbo 做出動作 (參數), 參數的詳細動作說明請參考官方文件: [請按此連結]頁面中 Built-in movement 一段.

8. 控制 Zenbo 的表情為(參數), 參數的詳細對應表情說明請參考官方文件: [請按此連結] 頁面中 Facial Expression 一段.

9. 隱藏 Zenbo 的表情畫面.

10. 控制 Zenbo 說 (Hi,你好 / 看這裡/ WoW / YA).

11. 控制 Zenbo 說 (輸入文字).

12. 控制 Zenbo (說話/ 音樂/ 鬧鐘/ 通知) 的音量 (大聲點 / 小聲點).

13. 控制 Zenbo 以 (L1/ L2/ L3/ L4/ L5/ L6/ L7/ L8) 的語速, 說 (輸入文字). L1為語速最慢, 依序遞增, L8為語速最快.

14. 註冊 Zenbo 準備聽 語句(一/ 二 / 三/ 四/ 五) 是 (輸入文字), 通常與 [15. 我要開始聽] 的積木一起使用, 需間隔一秒後才能使用語音辨識積木.

15. 控制 Zenbo 啟動語音辨識.

16. 當 Zenbo 聽到 語句(一/ 二 / 三/ 四/ 五) 時, 做某些事情. 需要先使用[14. 準備要聽]的積木.

17. 刪除所有在 Zenbo 已經註冊的語句.

18. 撥放網址為(輸入網址)的 Youtube 影片.

19. 撥放網址為(輸入網址)的音樂.

20. 顯示網址為(輸入網址)的圖片.

21. 撥放在 Google 雲端硬碟服務中的影片, 其網址為(輸入網址). 此檔案必須在 Google 雲端硬碟服務內, 預先設定開啟共用連結.

22. 撥放在 Google 雲端硬碟服務中的音樂, 其網址為(輸入網址). 此檔案必須在 Google 雲端硬碟服務內, 預先設定開啟共用連結.

23. 瀏覽在 Google 雲端硬碟服務中的圖片, 其網址為(輸入網址). 此檔案必須在 Google 雲端硬碟服務內, 預先設定開啟共用連結.

24. 瀏覽在 Google 雲端硬碟服務中的文件, 其網址為(輸入網址). 此檔案必須在 Google 雲端硬碟服務內, 預先設定開啟共用連結.

24. 當觸摸 Zenbo 的頭時, 做某些事.
