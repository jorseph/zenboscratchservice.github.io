ZenboScratchService SOP

1. Launch ZenboScratchService and there is an IP address if it work well (notice that the target Zenbo device and Host PC must connect to the same Wi-Fi AP):

2. Enter http://scratchx.org/, open Extension URL, paste url by https://zenboscratchservice.github.io/extentions/ZenboScratchServiceExtension.js and click on “Open” button on a Host PC:

3. Click on “I understand, continue” button and enter online Scratch:

4. Download some .sbx example project in the following links and load the .sbx example project into scratchx(notice that fill previous ZenboScratchService IP address (e.g., 192.168.11.2 ) to replace 192.168.0.1 default value):
