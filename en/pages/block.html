Scratch Blocks

The followings are all Zenbo Scratch v1.2 block instructions：

1. Set the variable of Zenbo’s IP and you could get the Zenbo’s IP in Zenbo Scratch app.

2. Control Zenbo with IP (INPUT IP) move (forward / backward) ( 0.25 / 0.5 / 0.75 / 1.0 / 1.25 / 1. 5/ 1.75 / 2.0) meter at the speed (slowly / normally / fast).

3. Control Zenbo with IP (INPUT IP) to stop moving..

4. Control Zenbo’s head at IP (INPUT IP) to turn ( left / right/ top / down) ( 0 / 15 / 30 / 45) degrees. The degree is absolute angle. If the Zenbo is up 15 degree and you choose the head up 15 degree block, Zenbo would not move.

5. Control Zenbo’s body with IP (INPUT IP) to turn ( left / right) ( 0/ 15/ 30/ 45/ 60/ 75/ 90/ 105/ 120/ 135/ 150/ 165/ 180)degrees.

6. Control Zenbo with IP (INPUT IP) to ( stop / move forward / turn left/ turn right) continuously.

7. Control Zenbo with IP (INPUT IP) to do built-in movement (#). About the movement corresponding to #, please reference the official document: [click here]. See “Built-in movement“ section..

8. Control Zenbo with IP (INPUT IP) to set expression(#). About the expression corresponding to #, please reference the official document: [click here]. See “Facial Expression“ section.

9. Hide Zenbo's expression screen.

10. Control Zenbo with IP (INPUT IP) to say ( Hi, welcome to zenbo scratch service / Look here / WoW / YA).

11. Control Zenbo with IP (INPUT IP) to say (INPUT STRING).

12. Control Zenbo with IP (INPUT IP) of stream ( music / alarm clock/ notification/ robot speaking).

13. Control Zenbo with IP (INPUT IP) to speak (INPUT STRING) using speak speed (L1/ L2/ L3/ L4/ L5/ L6/ L7/ L8). The L1 correspond to the slowest speech speed, L2 to L8 is in ascending order, and the L8 is the fastest speech speed.

14. Register the Dialog System to listen specific context ( 1 / 2 / 3 / 4 / 5) at Zenbo with IP (INPUT IP). Usually use together with the block “15. start listen context”.

15. Control Zenbo with IP (INPUT IP) to start listening.

16. Do something when Zenbo at IP (INPUT IP) listened context ( 1 / 2 / 3 / 4 / 5). Use together with the block “14. register the dialog system to listen specific context”.

17. Delete all registered context of dialog system at Zenbo with IP (INPUT IP).

18. Play the Youtube video at URL:(Input URL).

19. Play the online audio file at URL:(Input URL).

20. Show the online image file at URL:(Input URL).

21. Play the video file at Google drive, which sharable URL is (Input URL). The file should be set the sharable link previously.

22. Play the audio file at Google drive, which sharable URL is (Input URL). The file should be set the sharable link previously.

23. Browse the image file at Google drive, which sharable URL is (Input URL). The file should be set the sharable link previously.

24. Browse the document file at Google drive, which sharable URL is (Input URL). The file should be set the sharable link previously.
